#接口文档规范

### 1 后端返回数据结构
    格式：
    {
            success:true|false, //请求的状态
            data:[],            //返回数据，无数据时也返回空字符"" 或空数组[]
            msg:''              //【可选，返回的一些提示信息】
    }
    
###2 后端提供开发接口及所需参数（参数需要注释），返回数据字段也需要注释，调试阶段需完成接口自测，来保证接口正常。

       contextPath + "/unifiedPay/OrderLogAction/queryOrder.go"

|    *          |             |      |         | |
